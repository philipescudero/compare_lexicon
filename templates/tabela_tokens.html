Tabela de Tokens - BIRL

Guia completo dos elementos da linguagem

Nossos Tokens Poderosos!

Nome do Token Lexemas Expressão Regular Descrição
INICIO_PROGRAMA BORA `\bBORA\b` Início do programa
FIM_PROGRAMA BIRL! `BIRL!` Fim do programa
DECLARACAO_VAR MONSTRO `\bMONSTRO\b` Declaração de variável
ATRIBUICAO TASAINDODAJAULA `\bTASAINDODAJAULA\b` Operador de atribuição (=)
PRINT GRITA `\bGRITA\b` Imprime algo no console
IF CONFERE_AI `\bCONFERE_AI\b` Início de condição IF
ELIF CONFERE_MAIS `\bCONFERE_MAIS\b` Condição ELIF (else if)
ELSE OU_NAO `\bOU_NAO\b` Condição ELSE
WHILE TREINA ATÉ `\bTREINA ATÉ\b` Laço while
FUNC FICA GRANDE `\bFICA GRANDE\b` Declaração de função
CALL CHAMA `\bCHAMA\b` Chamada de função
BOOLEAN_VERDADEIRO VERDADEIRO `\bVERDADEIRO\b` Valor booleano verdadeiro
BOOLEAN_FALSO FALSO `\bFALSO\b` Valor booleano falso
OP_LOGICO E, OU, NÃO `\bE\b|\bOU\b|\bNÃO\b` Operadores lógicos (AND, OR, NOT)
OP_ATRIBUICAO_COMPOSTA +=, -=, *=, /= `\+=|\-=|\*=|\/\=` Operadores de atribuição composta
NUM_DECIMAL 3.14, 0.5, 123.0 `\b\d+\.\d+\b` Números de ponto flutuante
NUMERO 10, 100, 42 `\b[0-9]+\b` Números inteiros
STRING "Qual foi?", "BIRL!" `"[^"\\n]*"` Texto entre aspas duplas
OP_ARITMETICO +, -, *, / `[+\\-*/]` Operadores aritméticos
OP_RELACIONAL_OU_IGUALDADE >=, <=, ==, !=, >, < `>=|<=|==|!=|>|<` Operadores relacionais e de igualdade
PARENTESES_ABRE Coloca anilha `\bColoca anilha\b` Início de parênteses (abre um bloco ou lista de argumentos)
PARENTESES_FECHA Tira anilha `\bTira anilha\b` Fim de parênteses (fecha um bloco ou lista de argumentos)
VIRGULA , `,` Vírgula (separador)
DOIS_PONTOS : `:` Dois pontos (início de bloco)
COMENTARIO # Isto é um comentário `#[^\n]*` Comentários de linha
ID x, resultado, idade `\b[a-zA-Z_][a-zA-Z0-9_]*\b` Identificador (nome de variável ou função)
NEWLINE (quebra de linha) `\n` Quebra de linha (ignorada na análise)
SKIP (espaço, tab) `[ \t]+` Espaços em branco e tabulações (ignorados)
MISMATCH @, $, #, etc. `.` Qualquer caractere não reconhecido (erro léxico)
